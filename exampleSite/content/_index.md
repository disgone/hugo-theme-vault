Welcome to the Vault theme demo!

This is a minimalist, responsive Hugo theme designed for blogs and personal sites. Browse the latest posts below, or use the navigation menu to explore different sections of the site.

## Features

- Clean, minimalist design
- Responsive layout
- Built-in support for tags and categories
- Syntax highlighting
- Fast performance with Hugo Pipes for CSS/JS processing

## Getting Started

The content in this example site demonstrates the theme's features. Check out the blog posts to see how posts are displayed, or visit the About page to learn more.
