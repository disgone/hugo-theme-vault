# About Vault Theme

Vault is a minimalist, responsive Hugo theme designed for blogs and personal sites.

## Design Philosophy

The theme follows these principles:

- **Simplicity**: Clean, distraction-free design focused on content
- **Performance**: Fast loading times with optimized assets
- **Flexibility**: Easy to customize and extend
- **Accessibility**: Good contrast, readable typography, and keyboard navigation

## Features

- Responsive layout that works on all devices
- Built-in support for tags and categories
- Syntax highlighting for code blocks
- Fast page loads with Hugo Pipes
- Minimal JavaScript
- Clean, semantic HTML

## Technology

Built with:

- [Hugo](https://gohugo.io/) - Static site generator
- SCSS/Less - CSS preprocessing
- PostCSS - CSS transformations
- Webpack - Asset bundling (for theme development)
