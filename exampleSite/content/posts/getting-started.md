Welcome to your first post using the Vault theme! This post demonstrates the basic features of the theme.

## Installation

The Vault theme is easy to install. You can use it in three different ways:

1. **Self-contained**: Clone the theme and run the exampleSite directly
2. **Git submodule**: Add the theme as a submodule to your project
3. **Hugo modules**: Use Hugo's native module system

## Features

This theme includes:

- Clean typography
- Responsive design
- Built-in syntax highlighting
- Tag and category support
- Minimal configuration

## Customization

You can customize the theme by modifying the `config.toml` file. The theme supports various parameters for social links, analytics, and custom assets.
